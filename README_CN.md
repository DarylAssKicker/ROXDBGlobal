# ROXDBGlobal v2.0

ROXDBGlobal 是一个ROX资料查询平台，为玩家提供方便快捷的游戏资料查询服务。
基于ROX cn、jp、us版本构建，参考https://roxdb.cn网站资料。

## 🚦 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm >= 7.0.0 或 yarn >= 1.22.0
